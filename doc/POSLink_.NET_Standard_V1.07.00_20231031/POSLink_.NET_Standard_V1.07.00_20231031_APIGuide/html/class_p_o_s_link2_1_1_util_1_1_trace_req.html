◆ OrigRefNum

string POSLink2.Util.TraceReq.OrigRefNum
getset

The transaction record number in the terminal, required for follow up transactions after Sale/ Auth/ Return..., i.e. Void/PostAuth, terminal will prompt user to enter this if this field is NULL.

0 < Index <= 5000, normal data base.

5000 < Index <= 9000, stored data base for SAF.

9000 < Index <= 9999, failed data base for SAF.

Attribute: ans...32
