◆ AccountNumber

string POSLink2.FullIntegration.GetPinBlockReq.AccountNumber
getset

Account Number.

The length of the account number must from 13 to 19. It is required when the "PIN Algorithm" is set to 0(ISO9564_0) or 3(ISO9564_3).

This value should be masked PAN returned from the response of "InputAccountWithEmv" when the encryption is on.
