◆ FileName

string POSLink2.LogSetting.FileName
getset

Log file name. Default value is "POSLog".

The SDK will add date and suffix name. The real file name format is name+yyyyMMdd.txt.

◆ FilePath

string POSLink2.LogSetting.FilePath
getset

Log file path. The default path is current working directory.

Default file path is not support Xamarin.Android and Xamarin.IOS, please set their path.

For example:

Android : BaseContext.GetExternalFilesDir(null).Path

IOS: Environment.GetFolderPath(Environment.SpecialFolder.Personal)
