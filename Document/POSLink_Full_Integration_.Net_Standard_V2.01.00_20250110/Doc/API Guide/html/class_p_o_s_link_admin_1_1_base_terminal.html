Report status.

-1: No status returned.

0: Ready for card input.

9000005: Ready for Signature retry by pressing CLEAR key. (Same as 5)

9000006: Ready for PIN retry by inputting wrong offline PIN for EMV. (Same as 6)

9000007: Please See Phone

9000008: Please Try Another Card

9000009: Please Present One Card Only

9000010: Fallback Swipe

9020002: Ready for entering cashback.

9040003: Remove Card

9040010: Re-Insert Card

◆ UploadLog()

UploadResult POSLinkAdmin.BaseTerminal.UploadLog ( string[]  filePaths)
inline

This command can be used to upload the terminal logs. The log files will be uploaded to the file server specified by the terminal.

Note:

1. The terminal deletes the log files that have been successfully uploaded.

2. The terminal log files are packaged into one zip file and uploaded to the file server.

3. The terminal only logs necessary information for troubleshooting.

The disclaimers: https://www.pax.us/privacy-policy/.

Parameters
filePathsThe log names including path. If filePaths is null or empty, POSLink will upload all txt files from default path(working path). If you want files of your choice, please add log names including path to filePaths. The upload logs file should less then 30mb before compress.
Returns
UploadResult
