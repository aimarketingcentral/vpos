◆ AccountNumber

string POSLinkFullIntegration.FullIntegration.GetPinBlockRequest.AccountNumber
getset

Account Number.

The length of the account number must from 13 to 19. It is required when the "PinAlgorithm" is set to Iso9564_0 or Iso9564_3.

This value should be masked PAN returned from the response of "InputAccountWithEmv" when the encryption is on.
