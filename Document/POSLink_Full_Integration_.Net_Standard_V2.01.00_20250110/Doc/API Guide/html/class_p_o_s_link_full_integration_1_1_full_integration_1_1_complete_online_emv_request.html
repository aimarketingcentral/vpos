string POSLinkFullIntegration.FullIntegration.CompleteOnlineEmvRequest.ResponseCode
getset

Response Code. Host authorization response code, tag 8A.

i.e. Contact tag 8A:

Y1: Offline approved

Y3: Unable to go online, offline approved

Z1: Offline declined

Z3: Unable to go online, offline declined

Other: Online approved or declined

Attribute: ans2

◆ TagList

string POSLinkFullIntegration.FullIntegration.CompleteOnlineEmvRequest.TagList
getset

EMV tag list to be returned in EMV TLV data in response message.

Please see EMV Tags for Response of Authorize Card in Reference.

Example: Request for tag 0x9F26 0x9F27 0x4F, then this field would be: "9F269F274F".

Cannot request the following sensitive tags:

0x56: track 1 data

0x57: track 2 equivalent data

0x5A: PAN

0x9F1F: track1 discretionary data

0x9F6B: track 2 data(MasterCard Only)

0x9F20: track 2 discretionary data

Attribute: ans...1024
