◆ GetReportStatus()

override int POSLinkFullIntegration.Terminal.GetReportStatus ( )
inlinevirtual

POSLink reports the status.

Returns
Report status.

-1: No status returned.

0: Ready for card input.

1: Ready for PIN entry.

2: Ready for Signature.

3: Ready for Online Processing.

4: Ready for second card input.

5: Ready for Signature retry by pressing CLEAR key.

6: Ready for PIN retry by inputting wrong offline PIN for EMV.

9000000: Ready for card input. (Same as 0)

9000001: Ready for PIN entry. (Same as 1)

9000002: Ready for Signature. (Same as 2)

9000003: Ready for Online Processing. (Same as 3)

9000004: Ready for second card input (For REPLACE/MERGE). (Same as 4)

9000005: Ready for Signature retry by pressing CLEAR key. (Same as 5)

9000006: Ready for PIN retry by inputting wrong offline PIN for EMV. (Same as 6)

9000007: Please See Phone

9000008: Please Try Another Card

9000009: Please Present One Card Only

9000010: Fallback Swipe

9020002: Ready for entering cashback.

9040003: Remove Card

9040010: Re-Insert Card
