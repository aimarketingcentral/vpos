◆ EmvTlvData

string POSLinkFullIntegration.FullIntegration.CompleteOnlineEmvResponse.EmvTlvData
getset

EMV TLV data will be returned in consecutive TLV format data based on the Tag List filed in the request message.

Please see EMV Tags for Response of Authorize Card in Reference.

All the EMV TLV data shall be expanded to double length.

e.g. AID (A0 00 00 00 04 10 10): "4F07A0000000041010", Cardholder Name ("JACK"): "5F20044A41434B".

The following sensitive tags will NOT be included:

0x56: track 1 data

0x57: track 2 equivalent data

0x5A: PAN

0x9F1F: track1 discretionary data

0x9F6B: track 2 data(MasterCard Only)

0x9F20: track 2 discretionary data
