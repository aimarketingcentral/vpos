◆ LocalFailedReport()

ExecutionResult POSLink2.Report.Report.LocalFailedReport ( out LocalFailedReportRsp  response)
inline

Whenever a Settlement is run, there is the possibility that some of the transactions will fail during settlement. Those transactions are saved and their details can be retrieved using this message so that the merchant can directly settle the outstanding transactions with the card processor. If the record has reported, it will not be reported again.

It depends on host.
