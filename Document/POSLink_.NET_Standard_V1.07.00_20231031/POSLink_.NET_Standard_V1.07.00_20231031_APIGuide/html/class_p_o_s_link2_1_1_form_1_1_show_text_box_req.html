◆ InputText

string POSLink2.Form.ShowTextBoxReq.InputText
getset

Input Text.

0: No Input Text. (default)

1: Show Input Text.

If the input text is displayed, the three button will be automatically assigned to "Cancel", "Clear" and "Accept" state.

Input Text cannot be used with Signature Box.

Input Text needs an Input Text Title value to be used.

Attribute: n1

string POSLink2.Form.ShowTextBoxReq.InputType

Input type.

0: Alpha-Numeric string. (default)

1: Numeric string.

2: Date in MMDDYYYY format.

3: Time in HHMMSS format.

5: Password format.

7: Social security as "xxx-xx-xxxx".

◆ SignatureBox

string POSLink2.Form.ShowTextBoxReq.SignatureBox
getset

Signature box.

0: No signature box. (default)

1: Terminal Preference.

2: Display signature box horizontally.

3: Display signature box vertically.

The signature box should be displayed in the last page before the Button Key display.

If the signature box is displayed, the three buttons will be automatically assigned to "Cancel", "Clear" and "Accept" state.

Signature Box cannot be used with Input Text.

Attribute: n1
