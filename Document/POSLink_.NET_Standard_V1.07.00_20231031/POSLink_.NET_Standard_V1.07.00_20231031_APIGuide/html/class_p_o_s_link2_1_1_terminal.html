◆ BeginMultiCmdMode()

void POSLink2.Terminal.BeginMultiCmdMode ( )
inline

Begin Multiple Commands Mode.

In this mode, all operations will be executed after EndMultiCMDmode().

Not all operations support this mode. Please check return value of the operation.

In this mode, the return value is empty for every function which support MultiCMDMode.

◆ GetReportStatus()

int POSLink2.Terminal.GetReportStatus ( )
inline

POSLink reports the status during payment processing.

Returns
Report status.

-1: No status returned.

0: Ready for card input.

1: Ready for PIN entry.

2: Ready for Signature.

3: Ready for Online Processing.

4: Ready for second card input.

5: Ready for Signature retry by pressing CLEAR key.

6: Ready for PIN retry by inputting wrong offline PIN for EMV.

9020002: Ready for entering cashback.
