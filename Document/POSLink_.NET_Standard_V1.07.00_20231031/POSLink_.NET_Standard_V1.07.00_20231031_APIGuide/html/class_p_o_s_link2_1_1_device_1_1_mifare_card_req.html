◆ BlockValue

string POSLink2.Device.MifareCardReq.BlockValue
getset

Only valid when M1 Command type is:

1: write.The format is HEX and the length must be 32. For example, "01234567890123ABCDEF" means "\x01\x23\x45\x67\x89\x01\x23\xAB\xCD\xEF".

2: operate with increase value. The format is decimal and the max length is 9.

3: operate with decrease value. The format is decimal and the max length is 9.

Attribute: ans...32
