POSLink SDK introduction

PAX provides a flexible integration interface between the PAX payment device and the POS system, and removes the POS system from PCI and EMV loops. In semi-integrated configuration, the PAX device transmits data securely to host without POS system or ECR having to see any cardholder sensitive data. Responsibility of POS system is limited to communicating transaction request via POSLink API to the PAX device and printing the receipt. No sensitive data passes between the PAX device and the POS, thereby relieving the POS developer of responsibility for data security required in the credit card transaction.

Sample Code

POSLink2.CommSetting.TcpSetting commSetting = new POSLink2.CommSetting.TcpSetting();
commSetting.Ip = "127.0.0.1";
commSetting.Port = 10009;
commSetting.Timeout = 60000;
POSLink2.LogSetting logSetting = new POSLink2.LogSetting();
logSetting.FilePath = "D:\\";
//Default file path is not support Xamarin.Android and Xamarin.IOS, please set their path.
//Example:
//Android : BaseContext.GetExternalFilesDir(null).Path;
//IOS : logSetting.FilePath = Environment.GetFolderPath(Environment.SpecialFolder.Personal);
POSLink2.POSLink2 poslink = POSLink2.POSLink2.GetPOSLink2();
poslink.SetLogSetting(logSetting);
POSLink2.Terminal terminal = poslink.GetTerminal(commSetting);
POSLink2.Util.AmountReq amountReq = new POSLink2.Util.AmountReq();
amountReq.TransactionAmount = "100";
POSLink2.Util.TraceReq traceReq = new POSLink2.Util.TraceReq();
traceReq.EcrRefNum = "1";
POSLink2.Transaction.DoCreditReq doCreditRequest = new POSLink2.Transaction.DoCreditReq();
doCreditRequest.TransactionType = POSLink2.Const.TransType.Sale;
doCreditRequest.AmountInformation = amountReq;
doCreditRequest.TraceInformation = traceReq;
POSLink2.Transaction.DoCreditRsp doCreditResponse = null;
POSLink2.ExecutionResult executionResult = terminal.Transaction.DoCredit(doCreditRequest, out doCreditResponse);
if (executionResult.GetErrorCode() == POSLink2.ExecutionResult.Code.Ok)
{
//Transaction success.
}
